# FinTrust Online Banking

Enterprise-grade Online Banking System built using:

## Frontend

- Angular 21
- TypeScript 5.9.3
- Bootstrap 5
- Tailwind CSS
- Angular Material

## Backend

- ASP.NET Core 10
- Entity Framework Core
- Clean Architecture
- Microservices
- JWT Authentication
